## 基础教程
### 1. 标题
```
一级标题：# ……
二级标题：## ……
三级标题：### ……
四级标题：#### ……
五级标题：##### ……
六级标题：###### ……
```

### 2. 粗体
```
__ 或者 ** 包围
```

### 3. 斜体
```
* 或者 _ 包围
```

### 4. 删除线
```
～～包围
```

### 5. 代码和代码块
```
代码：`包围
代码块：```包围，在```后面可以写语言
```

### 6. 引用
```
> 你好
> > 你好
```

### 7. 列表
```
无序列表：- 或者 + 开头
有序列表：数字 + . 开头
```
